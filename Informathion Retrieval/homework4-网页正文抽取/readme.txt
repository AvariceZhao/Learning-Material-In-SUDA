2127405037赵鹏 信息检索作业四附件使用须知:

作业四标题: 网页正文抽取
要求:从html文件中抽取正文、锚文本和超链接并按指定格式输出到与html文件同名的文本文件中。
	
附件内容:homework4.py
	   readme.txt
*其中，homework4.py为本次作业主程序代码，作为文件的输入

python解释器版本 3.10.0

运行：
1.运行程序前需确保homework4.py所在路径下有名为webs的文件夹且文件夹中有至少一个html文件，否则程序将运行错误
2.运行前需要确保已安装BeautifulSoup4库和lxml解析器。如未安装，可打开cmd输入pip install beautifulsoup4和pip install lxml进行安装
3.可以使用python自带IDLE或pycharm打开并运行homework4.py代码进行网页提取
4.程序预计运行5-10s,当提取完毕后将显示“Completed!”,提取结果在网页所在的文件夹中
